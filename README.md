img2tant
========
Renders images into a set of UTF-8 sextant or octant blocks (black and white
only).

Usage
-----
```
Usage: img2tant [-hiv] [-o FILE] IMAGEFILE

  -6    force the use of sextant blocks
  -8    force the use of octant blocks
  -i    treat white pixels as black and black pixels as white
  -o    send output to file instead of stdout
  -h    print help and exit
  -v    print version and exit
```
